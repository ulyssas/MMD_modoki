# ビューポート見た目調整メモ

更新日: 2026-03-13

## 概要

2026-03-13 時点で、初期ビューポートの見た目と影の見え方を軽く調整した。
対象は toon 影の境界合わせ、非Toon材質の影色、床と空の初期色、未読込時の案内文。

## 反映内容

- Toon の光色を 100% 以上にしたとき、加算側の見え方が影の階調とずれにくいように、lit 判定を二値寄りではなく連続マスク寄りに調整した。
- Toon 色指定のない影ありオブジェクトは、`StandardMaterial` 側で ambient を確保し、黒潰れではなくライトグレー寄りの影になるようにした。
- 床はチェッカーを廃止し、無彩色の明るいグレーをベースにした。格子線はそのまま残している。
- 空と背景は無彩色のライトグレーに揃え、skydome と clear color の差が見えにくいようにした。
- モデル未読込時の中央案内は、「ファイル読込 か ドラッグ&ドロップしてください」に変更した。

## 主な変更ファイル

- `src/mmd-manager.ts`
- `src/x-file-loader.ts`
- `wgsl/toon_balanced_default.wgsl`
- `src/index.css`
- `src/timeline.ts`
- `src/i18n.ts`

## メモ

- 非Toon影色の変更は `.x` アクセサリと床の見え方に効く。既に読み込んでいるアクセサリは、必要なら再読込で確認する。
- 今回は見た目の微調整が中心で、機能追加ではない。
