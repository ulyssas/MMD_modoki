# v0.1.7 リリースノート下書き

更新日: 2026-04-03

`v0.1.7` は、影品質まわりのシェーダープリセット改善、`.x` アクセサリの見え方改善、背景画像 / 背景動画読み込み、DoF 焦点ターゲット選択を中心にした更新です。

## 主な追加・改善

### 1. 背景画像 / 背景動画の読み込み

- `ファイル読込` から背景画像を読み込めるようにした
  - 対応: `png` / `jpg` / `jpeg` / `bmp` / `webp`
- `ファイル読込` から背景動画を読み込めるようにした
  - 対応: `webm` / `mp4` / `avi`
- 背景メディアは project 保存 / 読み込みでパスを復元
- 上部 UI に背景表示の ON/OFF を追加
- 動画は音声なし、`0フレーム = 動画 0 秒` の前提で同期

補足:
- `avi` は Chromium / Electron 側の codec 対応に依存するため、環境によって再生できない場合があります

### 2. シェーダープリセット改善

- `Cel Shadow Sharp` を追加
  - 左下 1px toon 参照寄りの、硬いセルフ影境界を作りやすくした
- `Full Shadow` / `Full Light` を見直し
  - toon 未設定材質でも使いやすい全面影面 / 全面光面プリセットに調整
- `Light and Shadow` を no-toon 材質でも使いやすくした
- プリセット候補の並び順と表示内容を整理
  - 内部向け / 実験向けプリセットは一部非表示化

### 3. `.x` アクセサリ改善

- `.x` アクセサリ読込時に `Accessory Toon` を自動適用
- no-toon 材質向け fallback toon texture を導入
- `.x` ローダーで authoring 済み normal を優先利用
  - `MeshNormals` がある場合は再計算ではなく元法線を使用
- アクセサリ向けの ambient 補正と材質見え方を調整

### 4. DoF 焦点ターゲット選択

- 右パネル `FX > DoF` に `DoF中心モデル` / `DoF中心ボーン` を追加
- 初期値は `カメラ注視点`
- モデル選択時は `頭` 系ボーンを優先して既定候補にする
- DoF ターゲットは project 保存 / 読み込みに対応

### 5. UI 調整

- 上部パネルの配置を整理
  - `背景` ボタン位置調整
  - `FX` / `UI` / `Language` 周辺の順序整理
- エフェクト欄のシェーダー割り当てボタンを整理
  - `全材質へ割り当て`
  - `選択へ割り当て`
- 照明欄 `影` セクションの並び順を調整

## 既知の注意

- `.x` アクセサリの半透明影は、材質や texture alpha の組み合わせによってムラっぽく見える場合があります
- `avi` 背景動画は codec 依存のため、まずは `webm` / `mp4` の利用を推奨
- Linux 版は必要に応じて `--no-sandbox` を付けて起動案内する

## リリース確認メモ

- 背景画像の表示と project 復元
- 背景動画の表示 / 再生 / 向き / project 復元
- `Cel Shadow Sharp` / `Full Shadow` / `Full Light` / `Light and Shadow`
- `.x` アクセサリ読込時の自動プリセット適用
- `DoF中心モデル` / `DoF中心ボーン` の表示と追従
